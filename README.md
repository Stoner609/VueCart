# mycart

> A Vue.js project
> 簡易的購物車練習
> 增加：搜尋功能、不重複選取商品、再次點擊移除商品、清除購物商品

## Build Setup

``` bash
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
